# stringsplus

#### 介绍

stringsplus面向逆向工程的静态分析需求开发，以strings的输出结果作为输入，将IP、URL、目录、可执行文件、非垃圾字符串筛选出来，输出更有分析价值的结果。当strings生成的字符串数量特别大时，stringsplus至少能让静态分析不至于陷入无从入手的困境。

#### 使用说明


1. 需要在C盘建立一个名为re的目录
1. 输出结果文件将会保存在re目录中

> 命令行 > stringsplus filename.txt
