# stringsplus

#### 介绍

面向逆向工程的需求开发，将字符串信息分类整理，清洗strings生成的结果。当strings生成的字符串数量特别大（数十MB，甚至更多）时尤其适用。stringsplus以strings的输出结果作为输入，将IP、URL、Windows目录、可执行文件、非垃圾字符串筛选出来，输出各类信息汇总的文本文件。

#### 使用说明

`命令行 > stringsplus filename`
